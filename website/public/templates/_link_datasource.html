Start linking data sources

In order to start seeing your data on the QuantifiedDev dashboard, you need to start linking up data sources.

It's easy.

1. Download and install our plugins from here

Then

2. Click on "Go to QuantifiedDev Dashboard" in the plugin window or from the tools link in the toolbar (Not available yet on Visual Studio)

Or

2. Find your Stream Id and Read Token in the admin window of your plugin, enter them in the boxes and click the button below
